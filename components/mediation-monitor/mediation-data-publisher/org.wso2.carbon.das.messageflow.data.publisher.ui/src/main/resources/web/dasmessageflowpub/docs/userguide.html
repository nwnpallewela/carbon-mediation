Configuring Mediation Flow Data Publisher

  1. Go to ESB management console and select Configure→DAS Configuration from side panel.
  2. Fill the form.
    1. Setting up DAS Credentials.
      1. DAS URL : Enter the IP address of the DAS server. And the port should be the thrift port ( ex: tcp://127.0.0.1:7611 ).
      2. Username: Put the BAM Thrift log-in user name.
      3. Password: Put the BAM Thrift log-in password.
      4. Use server for publishing: Put a tick to enable publishing data to this server.
    2. Click on Save button.
